1. Working with a Function

In C programming, a function is a block of reusable code designed to perform a specific task. It helps organize code modularly and makes programs easier to maintain and understand. Functions can return a value or be void if they return none.

Syntax:

return_type function_name(parameter_list) {// body of the function}

Types of Functions:

  • Pre-defined library functions: Built-in functions such as printf() and sqrt(), found in headers like <stdio.h> and <math.h>.
  • User-defined functions: Custom functions written by programmers to perform specific tasks.

Key Components:

  • Function Prototype: Declares the function before use.Example: int add(int, int);
  • Function Call: Invokes the function to execute its task.Example: add(3, 5);
  • Function Definition: Contains the actual logic of the function.
    int add(int a, int b) {return a + b;}
  • Return Type: The data type of the value returned by the function. Use void if the function returns nothing.

4. Recursive Functions in C

A recursive function in C is one that calls itself to break down complex problems into smaller, more manageable sub-problems. The recursion continues until a base condition is met to terminate the process.

Syntax:


    return_type function_name(parameters) {
        if (base_condition)
            return value;
        else
            return function_name(modified_parameters);
    }

A. Factorial Program

This example computes the factorial of a number using recursion.


    #include <stdio.h>

    int factorial(int n) {
        if (n == 0) return 1;
        else return n * factorial(n - 1);
    }

    int main() {
        int n = 5;
        printf("Factorial: %d", factorial(n));
        printf("\n\nExecuted By 024NEB921");
        return 0;
    }

B. Fibonacci Program

This recursive program prints the nth number in the Fibonacci sequence.


    #include <stdio.h>

    int fibonacci(int n) {
        if (n == 0) return 0;
        if (n == 1) return 1;
        return fibonacci(n - 1) + fibonacci(n - 2);
    }

    int main() {
        int n = 7;
        printf("Fibonacci: %d", fibonacci(n));
        printf("\n\nExecuted By 024NEB921");
        return 0;
    }

5. Passing Arrays to Functions in C

In C programming, arrays can be passed to functions to perform operations such as printing or modifying elements. Here are two examples that demonstrate this concept:

A. Print Elements of an Array

This program passes an integer array to a function which prints each element.


    #include <stdio.h>

    void printArray(int arr[], int size) {
        for (int i = 0; i < size; i++)
            printf("%d ", arr[i]);
    }

    int main() {
        int nums[] = {1, 2, 3, 4};
        printArray(nums, 4);
        printf("\n\nExecuted By 024NEB921");
        return 0;
    }

B. Modify Elements of an Array

This program doubles each element in an array using a function.


    #include <stdio.h>

    void doubleArray(int arr[], int size) {
        for (int i = 0; i < size; i++)
            arr[i] *= 2;
    }

    int main() {
        int a[] = {2, 4, 6};
        doubleArray(a, 3);
        for (int i = 0; i < 3; i++)
            printf("%d ", a[i]);
        printf("\n\nExecuted By 024NEB921");
        return 0;
    }
